The Camp Arc

Five sessions. One arc.

CC Fest Coding Camp runs five sessions over five weeks. The sequence moves from first marks to open studio. Each session builds on the last, but every tool and sketch also stands alone. Jump in anywhere.

1 · Your Canvas, Your Voice Make your first marks and find your canvas voice
Coordinates Shape + Color Text RGB / HSB
⬡ Bridge: Color as Feeling
2 · Things That Move, Things That Listen Build things that move, react, and respond
Animation frameRate() map() lerp() If / Else
⬡ Bridge: Distance Becomes Behavior
3 · Patterns, Systems, and What They Say Let the code repeat, branch, and think for itself
For Loop Rows + Cols Functions Noise
⬡ Bridge: Noise vs Random
4 · Data as Material Turn numbers into pictures that mean something personal
Story Planner Data Mapper CSV / loadTable
⬡ Bridge: Data In, Drawing Out
5 · Open Studio Remix, share, reflect, teach
Image Remix Sound + Shape Postcard Game State Particles
⬡ Bridge: State Machines

    Best first

    Four friendly places to click when you just want to begin.

    How to use these Change one thing at a time. Predict before you move the slider. Try a value that breaks your guess — knowing what breaks is knowing how it works. You don't need to understand everything to make something. Unfinished work counts.
    Concept Bridges

    Understand the idea before you chase the syntax

    These are not output tools and not starter sketches. They are experiential explanations: interactive bridges between a fuzzy idea and the p5.js function or programming pattern behind it.
