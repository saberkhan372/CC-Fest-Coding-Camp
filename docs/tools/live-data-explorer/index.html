Try / Notice / Remix

Try this

Click "Fetch ISS Position" several times. The ISS moves at ~7.7 km/s — watch the latitude and longitude values change each time.

Notice this

CORS blocks direct API calls in many browsers. The mock data shows the exact same code pattern — swap the URL to go live when running on a server.

Remix this

Replace the ISS API with Open Meteo (free weather, no auth), or use a hardcoded array of objects and loop through it — the mapping code is identical.
