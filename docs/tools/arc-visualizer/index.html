Try this

Set start to 0° and drag stop from 0° to 360°. Watch the arc grow from nothing to a full circle.

Notice this

p5.js measures angles in radians. TWO_PI is a full circle (360°). PI is half (180°). The code panel shows the real radian values.

Remix this

Use arc() to make a pie chart slice, a Pac-Man shape, or a clock hand. Try changing w and h to make an elliptical arc.
