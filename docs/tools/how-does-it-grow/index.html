Teacher move

      

        

          Prompt
          Before reading code, ask learners to describe each track with a verb: count, rush, wave, drift, or chase.
        

        

          Misconception
          Students may think every changing number behaves like speed. Separate the number's raw value from the way it gets mapped onto the canvas.
        

        

          Ask
          Which expression would you use for a breathing shape? Which one for a score counter? Which one for a character easing toward the mouse?
