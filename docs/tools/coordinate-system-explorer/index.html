Open / Change / Predict / Remix / Teach

Use the prompts to turn the drawing canvas into a mini-lesson.

Open

Start with the grid visible and move your mouse around before drawing. Read the coordinate label like a map.

Change

Turn the grid on, choose Rectangle, and draw three different shapes while reading the coordinate labels.

Predict

The numbers in each shape function describe position and size. The same grid helps you predict where each shape will land.

Remix

Switch to Line mode and try drawing a letter, a map, or a tiny scene using only coordinates and shapes.

Teach

Ask someone else to point to a spot first, predict the x/y pair, then draw and compare the guess with the canvas.

Beginner-safe note

You can just watch the numbers move. Questions count as participation, and unfinished sketches are welcome.
