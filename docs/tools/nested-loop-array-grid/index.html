Nested Loop Pattern Studio

A compact seed for teaching how nested loops and spacing values build repeated grid patterns.

nested loops patterns grid

Live Seed

This one draws once in setup(), which makes it great for pausing and talking through the nested loop structure.

Try / Notice / Remix

Starter sketches stay lighter than full tools, but still carry teaching prompts.

Try this

Change spacing from 50 to 25 and notice how the grid gets denser.

Notice this

The outer loop steps across one direction of the grid. The inner loop fills the other direction.

Remix this

Replace rect() with circle(), or make the color depend on x and y.

Starter Code

Use this as a live-coding seed, not a finished tool.

// variables
let spacing = 50;

function setup() {
  createCanvas(400, 400);
  
  for (let x = 0; x < width; x = x + spacing) {
    for (let y = 0; y < height; y = y + spacing) {
      if (x % 100 == 0) {
        fill(random(255));
      } else {
        fill(random(255), random(0, 255), random(255));
      }
      rect(x, y, spacing);
    }
  }
}
