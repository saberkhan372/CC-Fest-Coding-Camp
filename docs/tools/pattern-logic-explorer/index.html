Try this

Notice this

The loop visits every cell. The condition decides what happens at each cell address. i is the row, j is the column.

Remix this

Type a custom condition: i * j % 5 === 0. Can you predict the pattern before pressing Enter? What about Math.pow(i-5,2) + Math.pow(j-5,2) < 16?
