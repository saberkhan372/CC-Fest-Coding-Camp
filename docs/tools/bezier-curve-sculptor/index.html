Break it on purpose: Drag a control point far off-canvas. The curve follows because Bezier handles extend infinitely.

    
Notice this: P0 and P3 are anchors on the curve. P1 and P2 are handles: they pull direction and tension.

  


  

    
Teacher move

    

      

        Prompt
        Drag only P1. What part of the curve changes first: the start, the end, or the middle?
      

      

        Misconception
        Control points look like they should be on the curve, but they pull it toward them without touching it.
      

      

        Ask
        Which points define where the curve begins and ends? Which points define how it leaves and arrives?
