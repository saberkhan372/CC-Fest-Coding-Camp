The input is the original number. In p5.js, this is often mouseX, mouseY, time, or a data value.

Output Control

View + Actions

Try this

Make the circle grow as your mouse moves right. Then reverse the output so it shrinks instead.

Notice this

map() does not make randomness. It converts one range into another range using a relationship you choose.

Remix this

Use mouseX for size and mouseY for color, or map data values to build a visual story.
