Session 4 — Before You Code

Data Story Planner

Five questions that turn a blank canvas into a plan. Answer them, copy your values into Data Mapper, then use the generated starter code when you are ready to build.

Open it Start with the canvas before the code. Change it Move one control or value at a time. Predict it Name what you expect before you run it. Remix it Turn the pattern toward your own sketch. Teach it Ask what changed and why.
1
What is your data?
Give your dataset a title and source, then type or paste 5–10 values. Numbers, or label:value pairs like Mon:6, Tue:8, Wed:5
Comma or newline separated. Words like high/medium/low are converted to 3/2/1.
2
What does each value represent?
What are you measuring, counting, or rating?
3
What visual property shows the value?
Pick one or two. Each one tells a different story with the same numbers.
4
What should the viewer notice?
A trend, a comparison, a feeling, an outlier, a question.
5
What might this hide?
Every visualization emphasizes something and leaves something out.
