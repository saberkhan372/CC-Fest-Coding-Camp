Try / Notice / Remix

Try this

Switch from BLEND to ADD with two bright colors. The overlap zone becomes brighter than either circle — RGB values literally add together.

Notice this

ADD makes overlapping areas brighter (adds RGB values). MULTIPLY makes them darker (multiplies normalized values). SCREEN is the inverse of MULTIPLY.

Remix this

Build a painting tool: replace the static circles with mouseX/mouseY, put blendMode() inside draw(), and let users paint overlapping strokes.
