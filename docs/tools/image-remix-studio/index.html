Key idea: in p5.js, an image is not only a picture. It can be drawn, repeated, tinted, cropped, sampled, masked, or treated as data.

Try this

Upload a photo, switch to pixel sample mode, and change the pixel size until the image becomes abstract but still recognizable.

Notice this

image() draws the whole image. get() samples color from one point. Both are useful for remixing images with code.
