World Coordinates vs Local Coordinates

A concept bridge for translate(), rotate(), push(), and pop(). The key idea: transformations move and rotate the coordinate system, not just the shape.

Presets:
The world grid is the canvas’s original coordinate system. Shapes use world coordinates until we transform the coordinate system.

Start with the world grid

The canvas begins with one coordinate system. x increases to the right. y increases downward. Shapes are placed using this world grid.

rect(300, 200, 80, 50)

Big idea

translate() moves the origin.

rotate() rotates the local grid.

push() saves the current coordinate system.

pop() restores the previous coordinate system.
