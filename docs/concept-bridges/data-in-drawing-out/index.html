The chart updates from your table

The p5.js code that would read this from a CSV file:

function preload() {
  table = loadTable("data.csv", "csv", "header");
}
function draw() {
  for (let i = 0; i < table.getRowCount(); i++) {
    let val = table.getNum(i, "Value");
    let h = map(val, 0, 100, 0, 300);
    fill(table.getString(i, "Color"));
    rect(x, height - h, barW, h);
  }
}
preload() gets external data ready before setup() and draw() try to use it.

Data can come from outside the sketch

Instead of typing every value directly into your code, p5.js can load a CSV table or JSON file. The sketch then uses that data to decide what to draw.

function preload() { table = loadTable("data.csv", "csv", "header"); }

Big idea

preload(): load files before the sketch starts.

Table: rows and columns, like a spreadsheet.

JSON: named data inside objects and arrays.

Visualization: map data values to visual properties.
