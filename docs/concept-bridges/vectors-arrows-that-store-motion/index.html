Two numbers working as one

A vector is an x and y value bundled together as a single object. It represents direction and magnitude at once. You can add vectors to combine forces, subtract them to find the direction from one point to another, normalize them to strip away magnitude while keeping direction, and scale them to set a precise speed. Every particle system, physics simulation, and steering behavior in p5.js is built on these four operations.
