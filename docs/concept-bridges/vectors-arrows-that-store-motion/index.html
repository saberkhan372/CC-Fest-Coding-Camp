A vector stores related numbers together. A position vector stores where something is: x and y in one value.

A vector stores x and y together

Instead of storing x and y as separate variables, a vector bundles them together. In p5.js, createVector(x, y) creates a vector.

let position = createVector(300, 200);

Big idea

Position: where something is.

Velocity: how position changes.

Acceleration: how velocity changes.

Vector addition: combine arrows to produce motion.
