Every image is a giant array

When you call loadPixels(), p5.js exposes the raw pixel data as a flat JavaScript array. Each pixel occupies four consecutive slots — red, green, blue, alpha. The formula to find a pixel at (x, y) is (y × width + x) × 4. Once you know this, images transform from opaque blobs into readable, writable data you can filter, generate, and paint at the lowest possible level.
