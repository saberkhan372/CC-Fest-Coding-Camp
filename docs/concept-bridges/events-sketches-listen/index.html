mouseX and mouseY are always available in draw(). They update continuously as the mouse moves.

Continuous input updates every frame

mouseX and mouseY are variables that p5.js updates automatically. draw() can use them every frame to make the sketch follow the pointer.

circle(mouseX, mouseY, 40);

Big idea

Continuous input: values available every frame, like mouseX.

Event function: runs once when something happens, like mousePressed().

Held input: checked every frame with keyIsDown().
