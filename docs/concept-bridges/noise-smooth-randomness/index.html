Smooth paths through a random-looking world

Perlin noise is a function that maps input numbers to smooth output values. Nearby inputs give nearby outputs. As you move through noise-space — whether by advancing a time counter or by using x/y position — the values flow rather than jump. This is what separates organic-looking terrain, creature movement, and texture from the harsh jitter of plain random().

Misconception Students often think noise() returns something random each call. Clarify: the same input coordinates always return the same output. It's a deterministic map, not a dice roll.
