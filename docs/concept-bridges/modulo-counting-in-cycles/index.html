Modulo: Counting in Cycles

A concept bridge for % (modulo). The remainder operator turns a forever-increasing counter into a repeating cycle — grids, stripes, alternating patterns, and looping animation.

Counting that wraps back to zero

Modulo is division's forgotten output — the remainder. When you count from 0 upward and take modulo N, you get cycles that repeat exactly every N steps. This is how frameCount becomes a looping animation timer, how an index becomes a repeating color, and how a tiling pattern knows which cell gets which shade. Grids, stripes, clocks, and loops all run on modulo.
