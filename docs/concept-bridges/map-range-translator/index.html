map() translates a number from one range into a matching position in another range.

The big idea

map() is a translator. It takes a number from an input world, figures out how far through that world it is, and places it in the same relative spot in an output world.

map(value, 0, 100, 20, 140)

Three-step mental model

1. Where is the input value between its minimum and maximum?

2. Turn that into a percent of the way through.

3. Put the output value the same percent through the output range.
