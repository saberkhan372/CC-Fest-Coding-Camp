Notice this

The code inside if runs when the condition is true. The code inside else runs when it is false.

Remix this

Use a condition to change color, size, background, movement, or whether a shape appears at all.

Break it

Drag the value slider to exactly match the threshold number. Does the condition show true or false? Now look at the generated code — does it use > or >=? Copy the code and flip the operator in your own sketch. What changes at that exact boundary value?
