Try / Notice / Remix

Try this

Set wander to 0 and seek to 0.15. Every agent locks onto the mouse instantly. Now raise wander back to 0.3 — they circle and drift rather than beeline.

Notice this

No agent knows about any other agent — each only reads its own position and the mouse. The flock-like pattern emerges purely from identical rules applied independently.

Remix this

Add a separation rule: if two agents are closer than 30px, push them apart. Each agent still acts alone, but now they respond to each other — true emergent behavior.
