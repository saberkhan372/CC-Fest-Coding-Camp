Try / Notice / Remix

Try this

Move from Step 1 to Step 6 watching both the code and the canvas. At each step more balls appear but the code gets shorter, not longer.

Notice this

At Step 5, each ball has its own x, y, speed, and color stored as object properties. At Step 6, the class packages that together as a reusable template.

Remix this

Add a size property to the class. Give each ball a unique size in the constructor. Add collision: if two balls overlap, reverse their speeds.
