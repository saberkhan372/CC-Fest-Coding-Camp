Try / Notice / Remix

Try this

Change the font URL to a different Google Fonts .ttf link. Visit fonts.google.com, pick a font, click "Download family", then find the direct CDN .ttf URL.

Notice this

preload() finishes before setup() runs — the font is always ready when the sketch starts. No waiting, no race conditions.

Remix this

Add a second font variable, load two fonts in preload(), and alternate between them. Or animate the font size using sin(frameCount * 0.05).
