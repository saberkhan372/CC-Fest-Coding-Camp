Dot Field Position Sketch

A simple array seed that stores many x/y positions, then redraws them as a single related system.

arrays coordinates repetition

Live Seed

This is a clean example of arrays as storage. The sketch keeps lists of positions, then loops through them to draw the field.

Try / Notice / Remix

Use this one when you want arrays to feel concrete and visual.

Try this

Change the number of points from 100 to 20 or 200.

Notice this

The two arrays stay connected by index: xPositions[i] belongs with yPositions[i].

Remix this

Change the dots into stars, faces, or different colors based on the position values.

Starter Code

Short enough to live-code, strong enough to remix.

let xPositions = [];
let yPositions = [];

function setup() {
  createCanvas(400, 400);
  noStroke();
  generateDotField();
}

function draw() {
  background(220);
  fill(0);
  
  for (let i = 0; i < xPositions.length; i++) {
    ellipse(xPositions[i], yPositions[i], 10, 10);
  }
}

function mousePressed() {
  generateDotField();
}

function generateDotField() {
  xPositions = [];
  yPositions = [];

  for (let i = 0; i < 100; i++) {
    xPositions.push(random(width));
    yPositions.push(random(height));
  }
}
