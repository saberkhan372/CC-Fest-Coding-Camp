Session 4 Starter Sketch

Random Poetry Generator

A tiny language sketch that uses arrays and randomness to build short poetic lines.

arrays randomness text

Live Seed

Click the canvas to generate a new tiny poem from the same word banks.

Try / Notice / Remix

This works well as a live-coding seed for generative language.

Try this

Add two new adjectives and two new nouns, then click again to see how the poem changes.

Notice this

The sketch stays short because the arrays do the heavy lifting. New words create new outcomes without changing the structure.

Remix this

Make it spooky, joyful, futuristic, or local to your neighborhood by swapping the word banks.

Starter Code

Simple enough to rewrite from scratch in front of a class.

let adjectives = ["baking", "blue", "wet", "lazy", "sunburnt"];
let nouns = ["tree", "hill", "highway", "sky", "city"];
let verbs = ["shimmers", "sweats", "melts", "swims", "gorges"];

function setup() {
  createCanvas(400, 400);
  noLoop();
  textFont("Helvetica", 48);
}

function draw() {
  background(255, 255, 100);
  text("the", 50, 100);
  text(random(adjectives), 50, 150);
  text(random(nouns), 50, 200);
  text(random(verbs), 50, 250);
}

function mousePressed() {
  redraw();
}
