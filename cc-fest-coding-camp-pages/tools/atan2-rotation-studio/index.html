Try this

Move your mouse in a slow circle around the center arrow. Watch how the angle reading climbs from -π to π and wraps around.

Notice this

atan2(dy, dx) always takes dy first, then dx. The result is in radians — p5.js uses radians by default for rotate().

Remix this

Replace the arrow with a spaceship, eye, or flower that always faces the mouse. Add + HALF_PI to the angle if your shape's nose points up instead of right.
