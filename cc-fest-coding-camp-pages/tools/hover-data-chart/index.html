Try / Notice / Remix

Try this

Drag the Sleep slider to 100 and Reading to 5. Hover between them to compare — the tooltip shows exact values not visible from bar height alone.

Notice this

The hover check uses mouseX > barX && mouseX < barX + barW. You're testing whether the mouse is inside a rectangular region defined by the same variables that drew the bar.

Remix this

Load real data using loadTable() instead of hardcoded values. Or change bar colors based on value — green for high, red for low — to add a second data layer.
