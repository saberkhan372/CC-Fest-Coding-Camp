Try / Notice / Remix

Try this

Set duration to 240 frames and watch the ease-in ball — it barely moves for the first half, then rushes to catch up at the end.

Notice this

The lerp ball never truly stops — it always moves a fraction of the remaining distance. That "almost there" behavior is the signature of lerp-based easing.

Remix this

Add a fifth row using lerp(x, target, noise(t)) — the amount itself varies over time, making the ball wander unpredictably toward the goal.
