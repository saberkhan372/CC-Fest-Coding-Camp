Try / Notice / Remix

Try this

Set lifespan to 30 and burst to 20, then click rapidly. The array fills and empties so fast you can feel the lifecycle running at full speed.

Notice this

The filter step particles = particles.filter(p => p.isAlive()) removes dead objects each frame. Without it, the array grows forever and the sketch slows down.

Remix this

Add a color property to the class. Make young particles one color and old particles another — the object itself carries the data needed to decide.
