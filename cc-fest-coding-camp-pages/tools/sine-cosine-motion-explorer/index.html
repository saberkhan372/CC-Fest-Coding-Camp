Try / Notice / Remix

Try this

Set amplitude to 120 and frequency to 0.08. Then drag phase and watch the whole wave slide sideways without changing its shape.

Notice this

Amplitude controls height, frequency controls how tightly the wave is packed, and phase shifts the starting point. All three are independent of each other.

Remix this

Use both sin() and cos() together to make the dot move in a circle. Set x = cos(t) * r and y = sin(t) * r.
