The Four Rules

< 2 neighbors → cell dies (underpopulation)
2–3 neighbors → cell survives
> 3 neighbors → cell dies (overcrowding)
Exactly 3 neighbors → dead cell is born

Try / Notice / Remix

Try this

Clear the grid. Draw this glider (use Step to watch it move): row 0: col 1. row 1: col 2. row 2: cols 0,1,2. Press Play.

Notice this

Each cell only checks its 8 neighbors — purely local rules. Yet gliders, oscillators, and still-lifes emerge from those same 4 rules applied across the whole grid.

Remix this

Change the survival rule: cells survive with 2, 3, or 4 neighbors. That one-number change produces completely different behavior (Highlife rules).
