Key idea: sound becomes a value like level. Then map(level, 0, 1, minSize, maxSize) turns that value into a visual change.

Try this

Start with simulated sound, then switch visual modes. Notice how the same sound level can control size, bars, particles, or color.

Notice this

Sound is being measured as numbers. Once you have numbers, you can use variables and map() to control anything visual.
