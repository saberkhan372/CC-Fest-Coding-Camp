Teacher move

    

      

        Prompt
        Pause the sketch. Drag the scale slider from one end to the other. What flips in the grid — and what stays the same?
      

      

        Misconception
        Students assume noise() is random every frame. Clarify: same scale + same time position = same value. Speed moves you through noise-space; it doesn't re-roll the dice.
      

      

        Ask
        Set speed to 0. Click Step. What changed? What does that tell you about how time works in noise()?
      

    

    
Break it on purpose: Set scale to its maximum. Notice how the grid looks almost random? That's the edge of the smooth zone — inputs are so far apart they lose their relationship.
