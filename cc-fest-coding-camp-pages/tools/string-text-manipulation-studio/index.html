Try / Notice / Remix

Try this

Type a sentence like the quick brown fox. Then set slice to characters 4–9 and watch only those characters appear on the canvas as a new word.

Notice this

split(" ") breaks the string on every space and returns an array — each word becomes an element. join(" ") does the reverse, collapsing the array back into a string.

Remix this

Use split() to break text into letters, then draw each letter at a different position or size on the canvas — turning a word into a visual composition.
