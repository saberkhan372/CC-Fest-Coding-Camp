Creative coding tools that feel like one workshop, not seventeen separate files.

A CC Fest library for learning shapes, motion, loops, arrays, interaction, stories, objects, and 3D through playful p5.js experiments. Start with polished teaching tools, then branch into starter sketches for live coding and remixing.

What learners should feel here

  • I know what this tool teaches.
  • I can change something safely.
  • I can connect the controls to the code.
  • I can remix the idea into my own sketch.

The CC Fest arc

Move from drawing single shapes to building systems, telling stories with data, and making interactive worlds. The homepage now mirrors that learning journey so teachers and participants can choose tools intentionally.

1. ShapesCoordinates, parameters, color
2. MotionVariables, animation, events
3. SystemsLoops, arrays, patterns
4. StoriesText generation, data, visualization
5. ExtensionsCollisions, objects, WEBGL
