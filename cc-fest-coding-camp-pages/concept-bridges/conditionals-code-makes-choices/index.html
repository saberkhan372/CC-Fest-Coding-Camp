Conditionals: Code Makes Choices

A concept bridge for if, else if, else, comparison operators, Boolean values, and combining conditions. A conditional lets the sketch choose what happens based on the current situation.

Presets:
A condition is a question with a true or false answer. If the answer is true, the code inside runs.

if asks a true/false question

An if statement checks a condition. If the condition is true, the code inside the curly braces runs. If it is false, that code is skipped.

if (x > 400) { fill("red"); }

Big idea

Condition: a true/false question.

Comparison: checks values with >, <, ===, etc.

Branch: the path the program takes.

AND / OR: combines multiple conditions.
