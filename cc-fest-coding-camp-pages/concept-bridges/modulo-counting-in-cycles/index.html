Modulo turns counting into cycles

The % operator gives the remainder after division. If you count with % 4, the result cycles through 0, 1, 2, 3, 0, 1, 2, 3...

0 % 4 = 0
1 % 4 = 1
2 % 4 = 2
3 % 4 = 3
4 % 4 = 0

Big idea

Counting keeps going up.

Modulo wraps the count back into a smaller set of remainders.

Remainders can control color, timing, animation, and pattern.
