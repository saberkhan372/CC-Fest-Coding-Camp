Distance Becomes Behavior

A concept bridge for dist(). Distance turns “how close?” into a number. Once closeness is a number, your sketch can use it for size, brightness, collision, unlocking, ripples, and interaction.

Distance is input for decisions and feeling

Once you have let d = dist(...), your sketch can react to the world. That one number can become scale, brightness, collision, a locked door, or a whole ripple field. The geometry stays the same while the meaning changes.

Teacher move

Prompt Switch to the Size tab. Move the pointer to the exact center of the target — what does dist() return? Now predict what the shape size will be when you reach the outer edge of the range ring, before you move there.
Misconception Students think dist() is only for collision detection ("did the circles touch?"). The key insight is that dist() returns a continuous number — any value along the whole range — that can drive size, brightness, opacity, or speed proportionally, not just on/off.
Ask In Door Lock mode the door is binary — open or closed. How would you rewrite that behavior so the door slides open gradually as you approach, instead of snapping? Where in the code would the distance number feed into that calculation?
