Loose variables describe one thing

At first, it is common to store a shape's x, y, size, and speed in separate variables. That works, but the relationship between those variables is only in your head.

let x = 200; let y = 200; let size = 72; let speed = 1.6;

Big idea

Properties are what an object knows.

Methods are what an object does.

Classes are blueprints for making many similar objects.
