A picture is a grid

Digital images are made of pixels. Each pixel is a tiny square with a position, like (x, y), and color values.

pixel at (x, y) color = [red, green, blue, alpha]

Big idea

Pixel: one tiny colored square.

Image: a grid of pixels.

RGB: red, green, and blue number values.

Remix: read pixels, change values, draw something new.
