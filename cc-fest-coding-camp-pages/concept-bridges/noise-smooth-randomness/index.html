Noise: Smooth Randomness

A concept bridge for noise(). random() jumps because each value is independent. noise() changes smoothly because nearby inputs produce nearby outputs.

random jumps, noise flows

Scale presets:
random() forgets the previous value. noise() changes smoothly as its input changes.

random() jumps. noise() flows.

random() gives unrelated values. noise() gives values that are connected to nearby values, which makes it useful for natural-looking motion and textures.

random(0, 1)
noise(x)

Big idea

random() forgets the last value.

noise() remembers the neighborhood.

Small input changes create small output changes.
