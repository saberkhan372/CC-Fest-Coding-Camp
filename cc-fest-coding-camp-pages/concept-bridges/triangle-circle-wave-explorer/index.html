1. Start with a triangle

In a right triangle, sine and cosine compare side lengths. This first view stays in the 0°–90° range so the sides are easy to see as positive lengths.

cos(θ) = adjacent / hypotenuse
sin(θ) = opposite / hypotenuse

Big idea

Cosine tells us how far sideways.

Sine tells us how far up or down.

A wave is what happens when we record one of those values over time.

p5.js connection

let angle = radians(degrees);
let x = centerX + cos(angle) * radius;
let y = centerY - sin(angle) * radius;

In math, positive y goes up. In p5.js, positive y usually goes down, so this explorer subtracts sine to make the math view easier to read.
